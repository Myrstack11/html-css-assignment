# HTML & CSS Assignment

## Overview

Your task is to build a website about a topic of your own choosing using HTML and CSS.
The content is entirely up to you — it could be about a hobby, a sport, a game, a movie, an animal, or anything else you find interesting.

---

## Requirements

Your website must include all of the following:
- [ ] At least 3 pages (e.g., Home, About, Contact)
- [ ] A consistent navigation menu on all pages
- [ ] A unique design using CSS (not just the default browser styles)

### HTML

- [ ] A valid HTML document structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- [ ] A title for each page page (using the `<title>` tag)
- [ ] At least one heading (`<h1>`) and one or more sub-headings (`<h2>`, `<h3>`)
- [ ] At least one paragraph (`<p>`)
- [ ] At least one list (ordered `<ol>` or unordered `<ul>`)
- [ ] At least one image (`<img>`) with an `alt` attribute
- [ ] At least one link (`<a>`) that goes somewhere relevant

### CSS

- [ ] A separate `.css` file linked to your HTML file
- [ ] Styling for colors (background and/or text)
- [ ] Styling for fonts (font family, size, or weight)
- [ ] Styling for spacing (margin and/or padding)
- [ ] At least one **class selector** (`.classname`)
- [ ] At least one **id selector** (`#idname`)

---

## Getting Started

1. **Fork or clone** this repository to your own GitHub/GitLab account.
2. Create your `index.html` and `style.css` files.
3. Build your site step by step — start with the HTML structure, then add CSS.
4. Commit your changes regularly with clear commit messages.
5. Make sure your final version is pushed to your repository before submitting.

---

## Submitting

Share the link to your Git repository when you are done.

---

## AI Usage Policy

You are allowed to use AI tools (such as ChatGPT, GitHub Copilot, etc.) to help you with this assignment.

However, the following rules apply:

- **You must understand everything you submit.** If you use AI-generated code, make sure you can read it, explain it, and answer questions about it. You may be asked to explain your code during or after the assignment.
- **If you used AI, you must document it.** Add a section at the bottom of your own `README.md` called `## AI Usage` that describes:
  - Which tool(s) you used
  - What you asked it to help with
  - Why you chose to use it for that part

### Example

```markdown
## AI Usage

I used ChatGPT to help me write the CSS for centering my navigation menu.
I was unsure how flexbox worked, so I asked for an explanation and an example.
I then modified the code to fit my own design and made sure I understood each property used.
```

Using AI without documenting it, or submitting code you cannot explain, is not acceptable.

## Tips

- Test your page in a browser as you go — save the file and refresh.
- Use comments in your code (`<!-- HTML comment -->` / `/* CSS comment */`) to explain what you are doing.
- Keep your code clean and indented so it is easy to read.
- If you get stuck, search for help on [MDN Web Docs](https://developer.mozilla.org).
